# Basketball Scoreboard

Simple web app to track scores for two teams.

## Features
- Add +1, +2, +3 points for each team
- Reset both scores
- Clean UI with Flexbox

## Files
- `index.html` → structure
- `styles.css` → styling
- `index.js` → logic

## Run
Open `index.html` in your browser.

Changes have been made to the headers
